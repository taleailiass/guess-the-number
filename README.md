# guess-the-number (C)
a fun and interactive console-based number guessing game written in C
This is a console-based C game where the computer picks a rando, number , and your task is to guess it.
ths game includes : 
-Multiple difficulty levels ( Easy, Medium , Hard) 
-Limited difficultty per level
-Hints to guide  your guesses 
-Score tracking 
-Replay option
This project is to practice C programming concepts like loops,conditions,functions,and user inputs .
